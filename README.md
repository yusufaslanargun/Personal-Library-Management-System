# Personal Library Management System (PLMS)

Personal Library Management System for organizing books and DVDs with inventory management, progress tracking, and external data integration.

This repository currently focuses on project documentation (RAD and SDD).

---

## Contents

### Requirements Analysis Document (RAD)
Project scope, goals, stakeholders, functional/non-functional requirements, constraints, and success criteria.  
File: `Personal Library Management System – RAD.docx`

### Software Design Document (SDD)
Architecture, modules/components, data model, core flows, and design decisions.  
File: `Personal_Library_Management_System–SDD.docx`

---

## System Scope (High-Level)

PLMS aims to provide:

- Inventory / Catalog Management
  - Manage a personal collection of books and DVDs
  - Keep item records consistent and searchable

- Progress Tracking
  - Track reading/watching progress over time

- External Data Integration
  - Enrich library entries via external data sources (e.g., metadata lookups)

For detailed requirements and design, refer to the RAD and SDD documents.
